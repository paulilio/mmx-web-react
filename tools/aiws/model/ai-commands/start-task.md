# start-task

## Description
Iniciar uma task. Ler o contexto do projeto e entender o que precisa ser feito.

## Steps
1. Ler o Context Kernel na ordem: .ai/AGENTS.md → SYSTEM.md → CODEBASE_MAP.md → CONTEXT_SURFACES.md
2. Ler a task em .aiws/tasks/tk-XXX.md
3. Resumir para o usuario: Objective, Plan, Code Surface
4. Perguntar se pode prosseguir ou se ha ajustes

## Output
Resumo da task e confirmacao do usuario para iniciar.
