# task-plan

## Description
Criar ou refinar o plano de execucao da task.

## Steps
1. Analisar o Objective e Context da task
2. Consultar CODEBASE_MAP.md para identificar arquivos relevantes
3. Consultar CONTEXT_SURFACES.md para mapear impacto
4. Definir ou refinar o Plan com etapas claras e checklist
5. Atualizar Code Surface da task com os arquivos identificados
6. Apresentar o plano ao usuario para aprovacao

## Output
Task atualizada com Plan e Code Surface definidos.
